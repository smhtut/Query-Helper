# Query-Helper
- Query Helper is a java mini project.
- It can be used as a tiny template engine.
- It can help developers to write database queries especially.
- Ctrl + f = Formula Text
- Ctrl + i = Input Text
- Ctrl + r = Click 'Run' button
- Ctrl + o = Output Text
