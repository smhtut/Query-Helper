# Query-Helper
Description
- Query Helper is a java mini project.
- It can be used as a tiny template engine.
- It can help developers to write database queries especially.

Requirements
- JDK 1.8+

User Manual
- Download and click a "Query_Helper_Version_1.2".jar which is a runnable jar file.

Short cut
- Ctrl + f = Formula Text
- Ctrl + i = Input Text
- Ctrl + r = Click 'Run' button
- Ctrl + o = Output Text
